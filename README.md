# Nawahi Frontend

This is the frontend for Nawahi — a full-stack service marketplace. Built using React and TailwindCSS.

## 🧪 Stack

- React 18+
- TailwindCSS
- Axios
- React Router

## 📄 Pages

- Homepage
- Login / Register
- User Dashboard
- Product Listing / Details
- Consultation form

## 🛠️ Getting Started

```bash
git clone https://github.com/hamdyadam97/nawahi-frontend.git
cd nawahi-frontend
npm install
npm start
